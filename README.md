# COVID-19 Global Data Analysis and Forecasting

## Overview

This project performs a comprehensive analysis of global COVID-19 data sourced from
Our World in Data. It covers data cleaning, exploratory data analysis, interactive
geospatial visualization, and machine learning based case forecasting. The project
is structured across three Jupyter notebooks, each handling a distinct phase of the
data science pipeline.

## Notebooks

### Notebook 1 — Data Cleaning and Exploratory Data Analysis
- Removed non-country aggregate entries from the dataset
- Handled missing values using forward fill and backward fill per country
- Retained 19 key columns relevant to cases, deaths, vaccinations, and demographics
- Generated 5 static charts covering global trends, top affected countries,
  continental death rates, vaccination progress, and GDP vs mortality analysis

### Notebook 2 — Interactive Geospatial Visualization
- Built an interactive choropleth map of total cases per million by country
- Built a choropleth map of death rate percentage by country
- Created an animated monthly progression map showing case spread over time
- Visualized fully vaccinated population by country using a bubble map
- All maps exported as standalone HTML files

### Notebook 3 — Machine Learning Forecasting
- Engineered lag features (1, 7, 14 days) and rolling averages (7, 14 days)
- Split data chronologically with 80% training and 20% testing
- Trained and evaluated three models: Linear Regression, Random Forest,
  and Gradient Boosting
- Compared models on MAE, RMSE, and R2 score
- Generated a 30-day new case forecast for India using the best performing model

## Dataset

Source: [Our World in Data — COVID-19 Dataset](https://ourworldindata.org/covid-deaths)

The raw dataset file `owid-covid-data.csv` is not included in this repository
due to its size. Download it from the link above and place it inside the `data/`
folder before running the notebooks.

## How to Run

1. Clone this repository
2. Download the dataset and place it in `data/`
3. Open Jupyter Notebook via Anaconda
4. Run the notebooks in order: 01 → 02 → 03
